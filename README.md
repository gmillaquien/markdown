# Ignorar Markdown

Solo se  debe anteponer la \ antes de cualquier elemento  de Markdown y esta no tendra efecto en el documento.


\  barra invertida

\ `  acento invertido

\ *  asterisco

\ _  guión bajo

\ {} llaves

\ [] corchetes

\ () paréntesis

\ #  almohadilla

\ +  símbolo de suma

\ -  guión

\ .  punto

\ !  exclamación
